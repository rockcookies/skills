# 自定义代码生成模板（Fork 专有）

`github.com/rockcookies/go-gen` 在 `gen.Tmpl` 结构体中暴露了 18 个 `text/template` 模板，允许在运行时覆写生成器的任意代码生成逻辑，无需 fork 代码。

## Tmpl 结构体

```go
type Tmpl struct {
    Header                      string // 生成文件头部（package + import）
    DIYMethod                   string // 自定义接口方法实现
    CRUDGenericMethod           string // 泛型 CRUD 方法
    CRUDMethod                  string // 普通 CRUD 方法
    CRUDMethodTest              string // CRUD 方法单元测试
    DIYMethodTestBasic          string // 自定义方法基础测试
    DIYMethodTest               string // 自定义方法完整测试
    Model                       string // model struct 定义
    ModelMethod                 string // model struct 方法
    DefaultQuery                string // 全局 Q 变量定义
    QueryMethod                 string // 查询方法实现
    QueryMethodTest             string // 查询方法测试
    TableQueryStruct            string // 无 context 的查询结构体
    TableQueryStructWithContext string // 带 context 的查询结构体
    TableQueryIface             string // 查询接口定义
    TableGenericQueryIface      string // 泛型查询接口定义
    DefineGenericsMethodStruct  string // 泛型方法结构体定义
    DefineMethodStruct          string // 普通方法结构体定义
}
```

所有字段默认由 `gen.NewGenerator()` 从 `internal/template` 包初始化，修改任意字段只影响对应代码块的输出。

## 快速入门

```go
import "github.com/rockcookies/go-gen"

g := gen.NewGenerator(gen.Config{
    OutPath: "../dal",
    Mode:    gen.WithDefaultQuery | gen.WithGeneric,
})
g.UseDB(db)

// 覆写文件头模板，添加自定义 build tag
g.Tmpl.Header = `// Code generated by go-gen. DO NOT EDIT.
//go:build !ignore_autogenerated

package {{.Package}}

import (
{{range .Imports}}    "{{.}}"
{{end}})
`

g.ApplyBasic(g.GenerateModel("users"))
g.Execute()
```

## 常见场景

### 场景 1：在所有生成文件头添加版权声明

```go
originalHeader := g.Tmpl.Header
g.Tmpl.Header = `// Copyright 2026 Your Company. All rights reserved.
` + originalHeader
```

### 场景 2：自定义 Model struct 输出格式

覆写 `g.Tmpl.Model` 可以控制生成的 struct 定义，例如添加固定注解、修改 tag 格式等。模板变量参考 `internal/generate/QueryStructMeta`。

### 场景 3：修改 DIYMethod 以生成额外的 tracing 代码

```go
g.Tmpl.DIYMethod = `
{{range .Methods}}
func ({{$.DaoVar}} {{$.QueryStructName}}) {{.MethodName}}({{.GetParamInTmpl}}) {{.GetResultTmpl}} {
    // auto-inject tracing
    span, ctx := tracer.Start(ctx, "{{.MethodName}}")
    defer span.End()
    {{.Body}}
}
{{end}}
`
```

## 注意事项

- 模板语法为标准 `text/template`，变量由生成器在运行时注入
- 修改模板后建议先在非生产路径测试输出（`OutPath: "/tmp/gen_test"`）
- 各模板字段可独立覆写，未覆写的字段保持默认行为
- 默认模板定义在 `internal/template/` 目录下，可作为覆写的参考基础
